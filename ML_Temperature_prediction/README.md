# [Создание регрессионной модели для предсказания температуры сплава](https://github.com/PavelNedviga/Portfolio#%D0%BF%D0%BE%D1%80%D1%82%D1%84%D0%BE%D0%BB%D0%B8%D0%BE)
Металлургический комбинат решает оптимизировать свои производственные расходы и уменьшить потребление энергии при обработке стали. Данная задача может быть решена путем использования модели, предсказывающей температуру стали от ряда параметров.

## Используемый стек и навыки:
Matplotlib,    
Pandas,    
Seaborn,    
math,    
os,    
numpy,    
sklearn,    
lightgbm,    
hyperopt,    
машинное обучение

## Результат и выводы
Итоговое качество предсказаний `MAE = 5.92` для модели на базе `LightGBMRegressor`. Погрешность менее 6 градусов при температурах 1450-1550.
