# [Предсказание цены автомобиля](https://github.com/PavelNedviga/Portfolio#%D0%BF%D0%BE%D1%80%D1%82%D1%84%D0%BE%D0%BB%D0%B8%D0%BE)
Данная работа посвящена предсказанию стоимости подержанных автомобилей. Сервис по продаже автомобилей разрабатывает приложение для привлечения новых клиентов. В качестве исторических данных имеется следующая информация с сервиса: технические характеристики, комплектации и цены автомобилей, пробег.

Необходимо построить модель для определения стоимости.

В решении задачи важны:
- качество предсказания;
- скорость предсказания;
- время обучения (вероятно модель будет периодически обновляться).


## Используемый стек и навыки:
CatBoost,    
LightGBM,    
Pandas,    
Seaborn,    
re,    
numpy,    
sklearn,    
машинное обучение

## Результат и выводы
Поэтапно были проанализированы и заполнены пропуски, очищены данные

Для оценки качества моделей применена метрика RMSE. Цены автомобилей очень сильно варьируются в зависимости от марки. Цены предсказаны в евро

Наименьшего RMSE удалось достичь в модели LightGMB: 1481.
